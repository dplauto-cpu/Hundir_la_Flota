## Hundir_la_Flota
Creado con Python Versión 3.10
#### `Descripción`:
Este proyecto permite jugar contra el ordenador al juego Hundir la Flota. Esta versión solamente permite el juego de una persona contra el ordenador.
Es un archivo de código no sofisticado creado por una sola persona con el fin de practicar con el lenguaje Python. Con toda probilidad, es mejorable. No está sujeto a licencias.
